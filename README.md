## Data Organization

The data is organized as follows:

```
|--CORE4D_Real
    |--object_models
    |--human_object_motions
    |--allocentric_RGBD_videos
    |--egocentric_RGB_videos
    |--scenes
    |--camera_parameters
    |--action_labels.json
|--CORE4D_Synthetic
    |--object_models
    |--human_object_motions
```
